# 拼豆生成器组件使用说明

## 组件结构

```
perler/
├── PerlerGenerator.tsx  # 主生成器组件
├── PixelGrid.tsx         # 像素网格可视化组件
└── MaterialList.tsx      # 材料清单组件
```

## 功能特性

### 1. 图片处理
- 支持常见图片格式（JPG, PNG, GIF, WebP）
- 自动像素化和颜色量化
- 实时参数调整

### 2. 参数控制
- **像素尺寸** (20-200): 控制图案精细程度
- **亮度** (-100 到 100): 调整图片明暗
- **对比度** (-100 到 100): 调整颜色对比
- **颜色抖动**: 改善渐变效果
- **显示尺寸** (5-50px): 调整预览大小

### 3. 导出功能
- 导出 PNG 格式的拼豆图案
- 导出 TXT 格式的材料清单

## 使用方法

组件已集成到 `WorkspacePage`，用户访问 `/workspace` 路由即可使用。

## 技术实现

- 使用 Canvas API 进行图片处理
- 使用 Konva.js 进行高性能渲染
- 使用 CIEDE2000 算法进行精确颜色匹配
- 支持 Floyd-Steinberg 抖动算法
